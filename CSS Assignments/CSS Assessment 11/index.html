CSS Positioning Techniques

Static Positioning

This box uses static positioning, the default positioning in CSS.

Relative Positioning

This box uses relative positioning. It is offset relative to its normal position.

Absolute Positioning

This box uses absolute positioning. It is positioned relative to its closest positioned ancestor.

Fixed Positioning

This box uses fixed positioning. It stays fixed relative to the viewport.

Sticky Positioning

This box uses sticky positioning. It sticks to the top when you scroll past it.
